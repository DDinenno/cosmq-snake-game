# Cosmq JS Snake Game

This is an example website for <a href="https://github.com/DDinenno/cosmq-js">Cosmq JS</a>

## getting started

Clone this repo
> git clone git@github.com:DDinenno/cosmq-snake-game.git

Install
> npm i

## Scripts

Run dev server 
> npm run dev

Create production build
> npm run build

Preview 
> npm run preview
